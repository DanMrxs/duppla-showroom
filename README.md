# duppla showroom

A premium landing page for **duppla showroom** — curated fashion for the modern woman.

Built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion. Designed with the editorial minimalism of Toteme, COS, and Aritzia.

## Live Site

- **Cloudflare Pages:** [duppla-showroom.pages.dev](https://duppla-showroom.pages.dev)
- **GitHub Repo:** [github.com/DanMrxs/duppla-showroom](https://github.com/DanMrxs/duppla-showroom)

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Framer Motion (animations)
- Lucide React (icons)

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/` and ready for static hosting.
